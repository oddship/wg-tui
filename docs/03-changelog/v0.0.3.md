# v0.0.3

Adds in-app file transfer workflows through Warpgate.

## Included

- transfer action from the target browser
- selectable `rsync` or `scp` tool choice
- selectable upload or download direction
- editable transfer flags with sensible per-tool defaults
- conditional path hints based on transfer direction
- remembered transfer drafts during editing and across restarts
- updated docs and release metadata
