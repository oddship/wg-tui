# v0.0.2

Adds the first tunnel workflow release.

## Included

- in-app tunnel action from the target browser
- tunnel port form with remote and local port entry
- dedicated tunnel status page with close and reconnect controls
- interactive SSH tunnel startup so Warpgate approval flows can complete
- improved search-to-browse flow with filter-preserving escape behavior
- more visible top-of-screen mode and shortcut chrome
- updated docs and release packaging metadata
