# v0.0.4

Makes day-to-day target picking smoother and gives you better control over where `wgt` reads config and local state.

## Included

- recently used targets rise to the top of the list and show a compact `↺` marker
- `--config`, `--cache-path`, and `--help` flags
- platform-aware default config and cache locations
- local approval-link prompt for normal SSH connects with open once, always, or no
- configurable approval browser command
- visible SSH and tunnel command previews, with `c` to copy
- search, config, and header polish from the recent UI cleanup work
